AI Interfaces

An AI Interface is any system, tool, or platform that provides a gateway for humans to interact with, command, or leverage artificial intelligence. These interfaces are the bridges that allow us to tap into the capabilities of AI models and complex automated systems.

The spectrum of AI interfaces is broad and constantly evolving. It includes everything from simple conversational text boxes to sophisticated software agents integrated into professional workflows.

Tool Use: The Language of Action

Regardless of the interface, a fundamental concept for modern AI agents is tool use. An agent's capabilities are defined by the set of tools it can call. These tools are the verbs of the AI worldâ€”the concrete actions the agent can perform.

A tool can be as simple as a direct API call (e.g., github.create_issue()) or a higher-level action provided by an automation service. For example, a service like Zapier can provide a "tool" called send_slack_message that abstracts away all the low-level details of the Slack API. The agent only needs to know the name of the tool and what information to provide it.

Types of Interfaces

The following pages will explore the different ways humans can interact with AI agents and leverage their tools:

  • Conversational AIs (Chatbots), such as Claude.
  • Automation Connectors, like Zapier MCP, which expose thousands of tools from different platforms.
  • Command-Line Interfaces (CLIs), for technical users and developers (e.g., Gemini CLI).
  • Specialized AI Agents, which come pre-packaged with a sophisticated set of tools for a specific domain, like software engineering.
